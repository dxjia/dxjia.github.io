在gradle中为release版本配置签名

Programmer
  • APK
  • Android
  • Gradle

任何一个Android APK 发布之前都会进行签名，没有签名的APK是无法在Android device上进行安装和使用的，而且对于发布到Google Play上的同一个应用，自始至终必须使用同一个签名文件，所以必须保存好签名文件。本文介绍如何在build.gradle中为release版本配置签名文件，这样在打包release版本时可以自动进行签名。

签名文件
